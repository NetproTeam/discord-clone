# Discord Clone Client

## 선행작업

- Backend Build

## 사용 기술 스텍
- FE: react, webRtc, stomp

## Getting Started

1. 패키지 설치:

 ```bash
 npm install
 ```
  1-1. 만약 react 버전으로 인해 설치가 안될 때: 
    
 ```bash
 npm install -save react@^17.0.2 react-dom@17.0.2
 ```
  
2. Start the development server:

 ```bash
 npm start
 ```
